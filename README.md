# Reaction–Diffusion（Gray–Scott）— クリックで SEED

- **クリック / タップ**した場所に**種（SEED）**を注入して模様を起こします。
- CPU 実装（WebGL 不要）。

## 操作
- クリック／ドラッグ：SEED 注入
- 速度：1–8（1フレームの反復回数）
- F, k, Du, Dv, dt：各係数
- 強注入：中心に大きめの種
- リセット：初期化
- PNG保存：画像保存

### 推奨値
- F ≈ 0.032–0.040、k ≈ 0.060–0.066
- Du=0.16, Dv=0.08, dt=1.0 から調整

## PWA
- ホーム画面に追加でオフライン起動。反映されない時は SW/Cache を消去。
